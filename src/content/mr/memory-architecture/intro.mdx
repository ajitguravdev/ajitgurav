# **Internal Storage & Architecture**

# **प्रस्तावना**

Internal Storage and Architecture हे कोणत्याही आधुनिक संगणकीय प्रणालीचा आधारस्तंभ आहेत. हे दस्तावेज प्रणालीच्या कार्यक्षमतेसाठी आणि डेटा व्यवस्थापनासाठी आवश्यक असलेल्या Internal Storage मूलभूत संकल्पना आणि Architecture प्रकाश टाकतो. या विषयाचे सखोल आकलन, डेटा कसा साठवला जातो, ऍक्सेस केला जातो आणि प्रक्रिया केली जाते, हे समजून घेण्यासाठी महत्त्वाचे आहे. एक सुव्यवस्थित आर्किटेक्चर प्रणालीची गती, विश्वासार्हता आणि स्केलेबिलिटी निश्चित करते, ज्यामुळे हे दोन्ही घटक एकमेकांशी कसे जोडलेले आहेत हे स्पष्ट होते.

# **Memory Organization**

संगणक प्रणालीमध्ये Memory हे विविध स्तरांवर आयोजित केले जाते जेणेकरून प्रोसेसरला डेटा वेगाने मिळू शकेल. ही रचना पिरॅमिड सारखी असते, जिथे वरच्या स्तरावर वेगवान पण कमी क्षमतेची मेमरी असते
