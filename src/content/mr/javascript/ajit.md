JavaScript ही जगातील सर्वात लोकप्रिय Programming Language आहे. सुरुवातीला फक्त ब्राउझरमध्ये 'Interactivity' देण्यासाठी वापरली जाणारी ही भाषा आज Server-side (Node.js), Mobile Apps आणि अगदी AI मध्येही वापरली जाते.

## १. JavaScript म्हणजे काय?

जर आपण एका वेबसाइटला मानवी शरीराशी तुलना केली, तर:
- **HTML** हे शरीराचे हाडांचे सापळे (Structure) आहे.
- **CSS** हे शरीराचे सौंदर्य (Design) आहे.
- **JavaScript** हे शरीराची हालचाल आणि मेंदू (Logic) आहे.

### **उदाहरण:**
जेव्हा तुम्ही एखाद्या बटणावर क्लिक करता आणि 'Login Success' असा मेसेज येतो, तेव्हा त्यामागे JavaScript काम करत असते.

## २. व्हेरिएबल्स (Variables)

डेटा स्टोअर करण्यासाठी आपण `let`, `const` आणि `var` चा वापर करतो.

```javascript
let name = "Ajit"; // बदलता येणारा डेटा
const age = 25;    // न बदलणारा डेटा (Constant)

console.log("माझे नाव " + name + " आहे.");
```

## ३. फंक्शन्स (Functions)

एखादे काम वारंवार करण्यासाठी आपण फंक्शन्स लिहितो.

```javascript
function welcome(user) {
  return "नमस्कार " + user + ", JavaScript च्या जगात तुमचे स्वागत आहे!";
}

console.log(welcome("विद्यार्थी"));
```
